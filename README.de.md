# Träwelcross

[Bei Google Play downloaden](https://play.google.com/store/apps/details?id=de.traewelcross)

[English](/README.md) / <b>Deutsch</b>

Träwelcross ist eine von [Träwelldroid](https://github.com/Traewelldroid/traewelldroid) inspirierte App. Genau wie Träwelldroid ist Träwelcross eine App für [traewelling.de](https://traewelling.de). Im Gegensatz zu Träwelldroid, ist Träwelcross in Flutter geschrieben, dadurch ist es auf mehreren Platformen durch nur eine Codebasis verfügbar, daher der Name Träwel*cross*. Leider habe ich aktuell keine gute Lösung für iOS zu bauen, daher sind dafür auch keine Builds angeboten. Ich hab's aber mal getestet und konnte bestätigen, dass Träwelcross ohne Probleme läuft.
